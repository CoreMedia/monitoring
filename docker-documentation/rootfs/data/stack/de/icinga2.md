# Icinga2

Icinga2 prüft zyklisch die Verfügbarkeit von Services oder die Gültigkeit von SSL Zertifiaten.

Dazu werden mehrere Service Checks in C++ erstellt um eine geringe Latenz zu ermöglichen.
